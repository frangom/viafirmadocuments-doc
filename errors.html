1. Instalación

Hazelcast CANNOT start on this node

No se pueden realizar algunas conexiones. Existen problemas de comunicación con elementos externos a la aplicación.

Hazelcast CANNOT start on this node. No matching network interface found. 
Interface matching must be either disabled or updated in the hazelcast.xml config file.

Solución:

Para solucionar este problema, editar hazelcast.xml e incluir en el nodo interfaces el host actual donde se intenta arrancar el servicio, por ejemplo 192.168.10.90. 

<interfaces enabled='true'>
  <interface>192.168.0.*</interface>
  <interface>192.168.10.*</interface>
  <interface>localhost</interface>
</interfaces>

Inconsistencia versión DB

Error de inconsistencia entre la versión que se intenta arrancar con la versión del modelo de datos (DB), mostrando durante el arranque de la aplicación una traza en log como la siguiente:

ERROR c.v.m.s.listener.StartUpListener - Invalid Configuration. Unable to connect to DB
org.hanuman.framework.exception.ConfigurationProblemErrror: Invalid version of DB version scheme. Required '0.0.24' but present:'0.0.12'

Solución:

Averiguar versión del modelo de datos actual, para ello:

  • opción 1: si la aplicación está arrancada, un super-administrador puede acceder al panel de control > información del sistema. Ahí viene este dato.
  • opción 2: si la aplicación no está arrancada, se puede consultar mediante la query sobre DB que se muestra a continuación:

SELECT VALUE FROM CONFIGURATION;
